# Startup Strategy Council Agent Skill

Expert startup strategy council with 6 advisors for validation, PMF, business models, positioning, and growth.

## Advisors

- **Peter Thiel** - Differentiation, monopoly
- **Paul Graham** - Early stage, founder focus
- **Eric Ries** - Lean, validation
- **Marc Andreessen** - PMF, markets
- **Sahil Lavingia** - Bootstrapping
- **April Dunford** - Positioning

## Installation

### npx skills add (recommended)
```bash
npx skills add antonioc-cl/startup-strategy-council
```

### Claude Code
```bash
/plugin add https://github.com/antonioc-cl/startup-strategy-council
```

### Claude.ai
Upload the `startup-strategy-council` folder or add via skills panel.

## Usage

Activate when you need help with:

- Startup strategy and direction
- Product-market fit validation
- Business model design
- Positioning and messaging
- Growth strategies (bootstrap-friendly)
- Pivot decisions

### Example Prompts

```
Use startup-strategy-council to validate my idea
Use startup-strategy-council to find product-market fit
Use startup-strategy-council to position my product
Use startup-strategy-council to design my business model
```
