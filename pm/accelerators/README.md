# Accelerators Directory

This directory contains resources and directories related to startup accelerators, incubators, and venture capital programs.

## Files

- `ai-accelerators.md` - Directory of AI-focused accelerators and programs
- `ai-statistics.md` - Statistics and data about AI accelerators
- `canada-directory.md` - Directory of Canadian accelerators and programs
- `list-of-accelerators.pdf` - Comprehensive list of accelerators (PDF format)
- `vc-directory.md` - Directory of venture capital firms and programs

## Purpose

These files support the accelerator application tracking feature in the platform, helping founders discover and apply to relevant accelerator programs.
