# Events System Design Document

> **Purpose:** Enable startup founders to create, manage, and promote small-scale events (demo days, pitch nights, networking meetups) with AI-powered planning, sponsor outreach, and marketing automation.

## Goals

- Reduce event planning from weeks to hours
- AI-generated task lists, marketing copy, sponsor outreach
- WhatsApp AI agent for attendee communication
- Integrated sponsor/venue discovery via Gemini search
- Seamless 3-panel experience consistent with StartupAI patterns

### Edge Function Details

#### `event-wizard`
```
Input:
  - step: number (1-4)
  - event_data: partial event object
  - startup_context: startup profile

Output:
  - suggestions: field pre-fill suggestions
  - tips: contextual tips for this step
  - validation: field validation results
  - warnings: feasibility/conflict warnings

Model: claude-sonnet-4-5
Tokens: ~500 input, ~300 output
```

#### `event-plan-generator`
```
Input:
  - event: complete event object
  - event_type: demo_day | pitch_night | networking | workshop | webinar | etc.
  - timeline_days: days until event

Output:
  - event_id: created event UUID
  - task_count: number of tasks generated (50+)
  - tasks[]: tasks with:
    - title, description
    - category (venue, sponsors, marketing, logistics, speakers, content)
    - due_offset_days (days before event)
    - dependencies[]
    - priority (high, medium, low)
    - is_milestone: boolean
  - timeline: milestone summary

Model: claude-sonnet-4-5
Tokens: ~800 input, ~2000 output
```

#### `event-marketing`
```
Input:
  - event_id: event UUID
  - asset_type: social_post | email | banner | graphic | press_release
  - platform: twitter | linkedin | instagram | email | website
  - prompt: optional custom prompt
  - generation_type: social_posts | email_sequence | graphics

Output:
  - content: generated text content
  - image_url: generated image URL (if graphics)
  - posts[]: platform-specific posts (if social_posts)
  - emails[]: email sequence (if email_sequence)
  - scheduled_at: suggested schedule time

Model: claude-sonnet-4-5 (copy) + gemini-3-pro-image-preview (graphics)
Tokens: ~600 input, ~800 output
```
