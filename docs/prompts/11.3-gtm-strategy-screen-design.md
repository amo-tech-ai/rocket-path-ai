## Screen Purpose

**Go-to-Market Strategy Planning**

The GTM Strategy screen provides founders with an intelligent, sophisticated way to plan and execute their go-to-market strategy. The screen enables AI-powered ICP generation from the startup profile, intelligent channel prioritization based on business model and stage, and comprehensive GTM planning with timelines and tactics.

## Features

### ICP Generation

**AI Generation from Profile**
- Extract ICP data from startup profile using ContentGenerator
- Generate company characteristics from industry and stage
- Generate decision maker from target customer
- Generate pain points from problem statement
- Generate buying behavior from business model
- Display suggestions for user approval

**Generation Workflow:**
1. User clicks "Generate ICP from Profile"
2. AI extracts relevant data from startup profile
3. AI maps profile to ICP sections
4. AI generates detailed ICP suggestions
5. Right panel shows generated ICP
6. User reviews each section
7. User approves or edits suggestions
8. ICP is saved to GTM plan

### Channel Prioritization

**AI Channel Ranking**
- Analyze business model and stage
- Score each channel (ICP Fit × CAC Efficiency × Stage Appropriateness × Competition × Time to ROI)
- Rank channels by overall score
- Provide reasoning for each ranking
- Display ranked channels with scores

**Prioritization Algorithm:**
- ICP Fit (30%): How well channel matches ICP
- CAC Efficiency (25%): Cost-effectiveness of channel
- Stage Appropriateness (20%): Suitability for current stage
- Competition Level (15%): Competitive intensity
- Time to ROI (10%): Speed to see results
- Overall Score = Weighted average of all factors

**Integration Points:**
- Edge function: `generate_content` (for ICP generation)
- Edge function: `coordinate_workflow` (for channel prioritization)
- Model: gemini-3-pro-preview (complex reasoning)
- Features: Structured Output, Thinking Mode, Google Search

### Keyword Research (Content Channel)

**AI Keyword Generation**

- Display keywords with metrics

**Integration Points:**

## User Journey

### First-Time GTM Creation

**Workflow Steps:**
1. User opens GTM Strategy screen
2. Screen is empty (no ICP, no channels)
3. User clicks "Generate ICP from Profile"
4. AI extracts data from startup profile
5. AI generates detailed ICP suggestions
6. Right panel shows generated ICP
7. User reviews each ICP section
8. User approves or edits suggestions
9. ICP is saved to GTM plan
10. User clicks "Prioritize Channels"
11. AI analyzes business model and stage
12. AI scores and ranks channels
13. Right panel shows ranked channels
14. User selects top 3-5 channels
15. User adds channel details
16. User clicks "Generate Strategy"
17. AI creates complete GTM plan
18. User reviews and saves plan

**Approval Gates:**
- ICP suggestions require user approval
- Channel selection requires user confirmation
- Strategy generation requires user review
- All changes require save confirmation

### Ongoing GTM Management

**Workflow Steps:**
1. User opens GTM Strategy with existing plan
2. User views ICP and channels
3. User edits ICP or channels as needed
4. User clicks "Prioritize Channels" to re-rank
5. AI re-analyzes and updates rankings
6. User updates strategy based on new rankings
7. User tracks channel performance (if active)
8. User receives AI alerts for underperforming channels

### GTM Plan Data Structure

- Startup ID (foreign key)
- ICP data (JSON object)
- Channels array (JSON array)
- Timeline phases (JSON array)

### Data Flow

**ICP Generation Flow:**
1. Frontend calls GTM service
2. GTM service calls AI service
3. AI service calls ContentGenerator edge function
4. Edge function generates ICP from profile
5. AI service returns ICP suggestions
6. GTM service returns suggestions to frontend
7. Frontend displays suggestions in right panel
8. User approves or edits
9. Frontend saves ICP via GTM service

**Channel Prioritization Flow:**
1. Frontend calls GTM service
2. GTM service calls AI service
3. AI service calls WorkflowCoordinator edge function
4. Edge function analyzes and scores channels
5. Edge function returns ranked channels
6. AI service returns rankings to frontend
7. GTM service returns rankings to frontend
8. Frontend displays rankings in right panel

## Error Handling

### ICP Generation Errors

**Profile Not Found**
- Display message: "Startup profile not found. Please complete the wizard first."
- Show link to wizard
- Disable ICP generation button

**Generation Failure**
- Display message: "Unable to generate ICP. Please try again or fill manually."
- Show manual entry option
- Log error to monitoring

### Channel Prioritization Errors

**Prioritization Failure**
- Display message: "Unable to prioritize channels. Please try again."
- Show manual channel selection option
- Log error to monitoring

**No Channels Defined**
- Display message: "No channels defined. Add channels to prioritize."
- Show add channel button
- Guide user to channel selection
