```css
:root {
  /* Superside AI Creative palette */

  --color-text: #1a1a2e;
  --color-text-muted: #64748b;
  --color-accent: #6366f1;
  --color-accent-hover: #4f46e5;

  /* Typography */
  --font-display: 'Inter', system-ui, sans-serif;
  --font-body: 'Inter', system-ui, sans-serif;

  /* Spacing (4px base) */
  --space-1: 0.25rem;
  --space-2: 0.5rem;

  --space-6: 1.5rem;
  --space-8: 2rem;

  /* Transitions */
  --transition-fast: 150ms ease;
  --transition-normal: 250ms ease;
}
```

## 3. Typography Scale (Hi-Fi)

| Token | Size | Weight | Line Height | Use |
|-------|------|--------|-------------|-----|
| Display-1 | 48–56px | 700 | 1.1 | Hero headline |
| Display-2 | 36–40px | 600 | 1.2 | Section H2 |
| Heading-1 | 24–28px | 600 | 1.3 | Subsection H3 |
| Heading-2 | 18–20px | 500 | 1.4 | Card titles |
| Body-L | 18px | 400 | 1.6 | Lead paragraph |
| Body-M | 16px | 400 | 1.6 | Body copy |
| Body-S | 14px | 400 | 1.5 | Captions |
| Label | 12px | 500 | 1.4 | Section labels, uppercase |

**Emphasis:** Use `font-style: italic` for 1–2 key words in headlines (e.g. "_delivered at scale_").

## 4. Layout Grid

| Breakpoint | Max Width | Columns | Gap |
|------------|-----------|---------|-----|
| Mobile | 100% | 1 | 24px |
| Tablet | 768px | 2 | 32px |
| Desktop | 1280px | 12 | 32px |
| Wide | 1440px | 12 | 48px |

**Section padding:** 48–64px vertical, 24px horizontal (mobile) → 80–96px vertical, 48px horizontal (desktop).

## 5. Component Mapping (Lo-Fi → Hi-Fi)

| Lo-Fi Element | Hi-Fi Component | States |
|---------------|-----------------|--------|
| `[Book a demo]` | Button (primary) | default, hover, active, focus, loading |
| Capability pill | Badge/Chip | default, hover (if clickable) |
| Capability card | Card (image + text) | default, hover |
| Stat box | StatCard | default |
| FAQ row | AccordionItem | closed, open |
| Team card | ProfileCard | default, hover |
| Section label | Uppercase caption | — |
| `[IMG]` | Image | loading, loaded, error |

## 7. Hi-Fi Prototype Spec (Production-Ready)

### Screen Documentation Pattern

```tsx
/**
 * Screen: AI Creative Product Page
 * Route: /ai-creative
 * Flow: Home → AI Creative (standalone product page)
 * States: default, accordion-open, modal-open (Book a demo)
 * Breakpoints: mobile 375, tablet 768, desktop 1280, wide 1440
 * Interactions:
 *   - Sticky header with CTA
 *   - Capability pills (decorative or anchor links)
 *   - FAQ accordion expand/collapse
 *   - Book a demo modal (form: email, name, company size, company name)
 */
```

### Key Content (Real, Not Placeholder)

- **Hero:** "AI-powered creative, _delivered at scale_"
- **Stats:** $1.4M saved, 60% efficiency, 2× faster, 50%+ cost reduction
- **Capabilities:** 8 items (AI video, Character, Product visuals, Image enhancements, Translation, Avatar/voice, Production at scale, Brand imagery)
- **Team:** 5 members with bios

## 9. Fidelity Progression Summary

| Aspect | Lo-Fi | Hi-Fi |
|--------|-------|-------|
| Visual | Grayscale | Full palette |
| Content | Labels, short | Real headlines, stats |
| Images | [IMG] placeholders | Real Superside assets |
| Interactions | Click-through | Hover, accordion, modal |
| Typography | Mono/sans | Inter, italic emphasis |
| Testing focus | Structure, hierarchy | Usability, conversion |
