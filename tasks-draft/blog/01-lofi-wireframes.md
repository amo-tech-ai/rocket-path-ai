## Content Hierarchy (Priority Order)

1. **Featured report** — largest visual weight, first content below hero
2. **Top stories** — second tier, equal weight to each other
3. **More reports** — third tier, smaller cards
4. **CTA** — conversion action, dark contrast block

## Navigation Flows

```
/blog (BlogIndex)
  │
  ├── Click featured card → /blog/fashion-ai-2026
  ├── Click top story card → /blog/ai-adoption-by-industry
  │                        → /blog/ai-jobs-future-of-work
  ├── Click grid card → /blog/ai-in-ecommerce
  │                   → /blog/ai-startup-products
  │                   → /blog/ai-investment-hubs
  ├── Click "Start validating" CTA → /validate
  └── Click "See how it works" CTA → /

/blog/:slug (BlogPost)
  │
  ├── Report renders inside Header/Footer shell
  ├── Invalid slug → redirect to /blog
  └── Report CTA links → /validator, /blog
```

## Annotations

- `[Badge]` — clickable category label (outline style)
- `░░░` — gradient background placeholder (no real images)
- `[Icon]` — Lucide icon centered on gradient
- `→` — arrow indicating navigation/link
- `(dark bg)` — DarkCTASection dark background block
