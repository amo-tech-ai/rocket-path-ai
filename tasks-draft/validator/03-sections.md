

# 🏛 1️⃣ STRATEGIC DIMENSIONS SECTION

---

## 🎯 Objective

Present multi-dimensional startup scoring as a structured strategic evaluation grid — not a simple progress bar list.

---

## 🔷 Wireframe Concept

```
--------------------------------------------------------
STRATEGIC DIMENSIONS

Short insight summary

----------------------------------------------
| Problem Clarity       | Timing            |
| Score + bar + insight | Score + bar      |
----------------------------------------------
| Solution Strength     | Market Size       |
----------------------------------------------
| Competition           | Business Model    |
----------------------------------------------
| Team Fit              |                   |
----------------------------------------------
```

Two-column grid (desktop).

---

## 🔥 Design Prompt

```
Design a Strategic Dimensions section for a startup evaluation dashboard.

Structure:
- Section label (small uppercase)
- Clear headline
- Two-column grid layout (desktop)
- Each dimension in a structured module

Each module must include:
- Dimension title
- Right-aligned score (e.g., 85 / 100)
- Elegant horizontal progress indicator
- Short strategic insight line
- Subtle color grading based on score

Visual style:
- Clean white surface
- Thin divider lines
- Generous internal spacing
- Refined typography
- Consulting-grade aesthetic

Avoid:
Generic dashboard bars.
Overly thick progress tracks.
Cluttered layout.
```

# 📊 2️⃣ MARKET SIZE SECTION

---

## 🎯 Objective

Present TAM, SAM, SOM as a structured financial narrative, not stacked cards.

---

## 🔷 Wireframe Concept

```
--------------------------------------------------------
MARKET SIZE

Strategic context line

----------------------------------------
| Visual (Circle or Pie) | TAM Card   |
|                        | SAM Card   |
|                        | SOM Card   |
----------------------------------------

Short insight conclusion
```

Balanced layout with visual emphasis.

---

## 🔥 Design Prompt

```
Design a Market Size section for an AI investment dashboard.

Structure:
- Clear section label
- Headline
- One-line strategic context
- Two-column layout (desktop)

Left side:
- Elegant visual representation (TAM/SAM/SOM relationship)
- Minimalist circular or concentric diagram

Right side:
- Three financial snapshot cards (TAM, SAM, SOM)
- Large serif metric
- Small uppercase label
- Short 1–2 line explanation

Design tone:
Investment-grade.
Minimal.
Architectural.
High-end SaaS intelligence.

Avoid:
Overly bright pie charts.
Heavy shadows.
Stacked tall text cards.
```

# 🎨 GLOBAL STYLE GUIDELINES (For All Sections)

• 12-column grid (desktop)
• 80–120px vertical spacing between sections
• Thin divider lines between sections
• Serif headlines, sans body
• Tabular numerals for metrics
• Subtle hover elevation
• Controlled motion only

---

# 🎬 MOTION SYSTEM (All Sections)

* Fade-up on entry
* Progress bars animate left-to-right
* Quadrant points fade in sequentially
* Cards stagger subtly
* No bounce animations
* Ease-out only
