# 🏛 SECTION 1 — HERO (Investment Briefing Header)

---

## 🎯 OBJECTIVE

Design a premium AI startup evaluation hero that feels like:

• A strategic investment memo
• High-end SaaS intelligence dashboard
• Consulting-grade interface
• Sophisticated and authoritative

---

## 🔷 HERO WIREFRAME STRUCTURE

Desktop layout:

```
--------------------------------------------------------
Startup Name                         Circular Score

Industry • Category • Stage          Status Badge

Short strategic positioning statement

--------------------------------------------------------
```

---

## 🔥 HERO DESIGN PROMPT

```
Design a luxury hero section for an AI startup evaluation dashboard.

Layout:
- Two-column layout (desktop)
- Left: Startup name, industry metadata, short positioning statement
- Right: Circular score visualization (67 / 100) with status badge

Visual hierarchy:
- Large serif startup name (48–56px)
- Small uppercase industry label with letter-spacing
- Clear separation between identity and score
- Balanced whitespace and grid alignment

Score component:
- Thin circular progress ring
- Large serif number centered
- Smaller "/100" aligned beside number
- Status badge below (Caution / Strong / Risk)
- Color-coded tier system
- Subtle animated stroke on load

Design tone:
Luxury.
Sophisticated.
Strategic.
High-end SaaS intelligence.

Avoid:
Generic admin dashboard styling.
Heavy glow effects.
Overly saturated colors.
Card clutter.
```

---

## 🎨 HERO STYLE GUIDE

Typography:

* Startup Name: Elegant serif display
* Metadata: 12–14px uppercase sans
* Positioning statement: 18px refined sans

Color:

* Neutral warm background
* Emerald or indigo accent
* Amber for caution
* Muted red for risk

Spacing:

* 80–120px top padding
* 32px spacing between major blocks

Motion:

* Score ring animates smoothly
* Text fades up gently
* No bounce or flashy transitions

# 🏛 SECTION 2 — EXECUTIVE SUMMARY

---

## 🎯 OBJECTIVE

Transform the executive summary from a dense text block into:

• Structured strategic briefing
• Clear narrative + metrics
• Balanced editorial layout
• Investor-ready presentation

---

## 🔷 EXECUTIVE SUMMARY WIREFRAME

Desktop layout:

```
--------------------------------------------------------
EXECUTIVE SUMMARY

Short strategic verdict (bold highlight line)

--------------------------------------------------------

[ Left Column ]                      [ Right Column ]

Narrative analysis                   Financial Snapshot Cards

- Market overview                    TAM
- Revenue model                      LTV/CAC
- Unit economics                     ARR Scenario
- Key risk                           Break-even

--------------------------------------------------------
```

---

## 🔥 EXECUTIVE SUMMARY DESIGN PROMPT

```
Redesign the Executive Summary section into a structured, consulting-grade layout.

Create:
- Clear section label
- Strong verdict headline
- Two-column layout (desktop)
- Left side: strategic narrative text
- Right side: financial snapshot cards

Financial cards:
- Compact
- Large serif primary metric
- Small uppercase label
- 1–2 line explanation
- Subtle border and soft elevation
- Generous internal padding

Narrative column:
- Improved readability
- Better paragraph spacing
- Strategic emphasis on key data points
- Clear hierarchy between insight and detail

Design tone:
Investment memo.
High-end.
Confident.
Analytical.

Avoid:
Tall text blocks.
Dense paragraphs.
Overuse of boxed containers.
```

---

## 🎨 EXECUTIVE SUMMARY STYLE GUIDE

Section label:

* 12px uppercase
* Muted tone
* Letter-spacing 0.15em

Verdict headline:

* 24–32px serif
* Strong but not bold-heavy

Narrative:

* 16–18px
* 1.6 line height
* Clear vertical spacing

Cards:

* 24px padding
* 12px radius
* 1px soft border
* Balanced shadow
* Clear visual grouping
