## 4. Bubble Charts

**When to use:** Showing correlation between 2-3 variables with a third dimension (size). Best for exploratory data with many data points. Use when scatter plot needs a third variable.

**Real-world examples:** CO2 emissions per capita vs population by country, market opportunity mapping, competitive landscape.

## Chart Type Decision Guide

```
Need to show...
  |
  +-- Change over time? --> Stacked Bar (#3), Grouped Bar (#1)
  |
  +-- Ranking / comparison? --> Horizontal Bar (#1), Grouped Bar (#1)
  |
  +-- Parts of a whole? --> Donut (#2), Stacked Bar (#3)
  |
  +-- Correlation (3 vars)? --> Bubble Chart (#4)
  |
  +-- Single KPI progress? --> Gauge (#5)
  |
  +-- Process / framework? --> Wheel Diagram (#6)
  |
  +-- Hero stat / headline? --> Stat Card (#7)
  |
  +-- Multi-variable grid? --> Matrix / Table (#8)
  |
  +-- A vs B contrast? --> Comparison Panel (#9)
```
