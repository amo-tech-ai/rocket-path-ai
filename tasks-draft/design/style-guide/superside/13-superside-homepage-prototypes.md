## 2. Hi-Fi Color Palette

| Token | Value | Use |
|-------|-------|-----|
| Page BG | `#ffffff` | Default |
| Section alt BG | `#f8f9fa` | Alternating sections |
| Text primary | `#1a1a2e` | Headlines, body |
| Text muted | `#64748b` | Labels, captions |
| Accent / CTA | `#6366f1` | Buttons, links |
| Accent hover | `#4f46e5` | CTA hover |
| Border | `#e2e8f0` | Dividers, cards |
| Success / check | `#22c55e` | Comparison ✓ |

## 3. Typography (Hi-Fi)

| Element | Size | Weight | Notes |
|---------|------|--------|-------|
| Hero headline | 48–56px | 700 | Italic on "_creative team's_" |
| Section H2 | 36–40px | 600 | Italic on 1–2 key words |
| Section H3 | 24–28px | 600 | |
| Section label | 12px | 500 | Uppercase |
| Body / subhead | 18px | 400 | Hero subhead |
| Stat value | 36–48px | 700 | 500+, 94%, 6 months |
| Stat label | 14px | 400 | Muted |
| Card title | 18–20px | 500 | Service/work cards |
| Quote | 20–24px | 400 | Serif optional |

## 5. Component Mapping (Lo-Fi → Hi-Fi)

| Lo-Fi Element | Hi-Fi Component | States |
|---------------|-----------------|--------|
| `[Book a demo]` | Button (primary) | default, hover, active, focus, loading |
| Logo strip | TrustBadge strip | — |
| 3-pillar cards | ValuePropCard | default, hover |
| Stats | StatCard | default |
| Testimonial | TestimonialCard | default, carousel nav |
| Service card | ServiceCard | default, hover, expanded (accordion) |
| Comparison row | ComparisonTableRow | — |
| Work card | PortfolioCard | default, hover |
| CTA section | CTASection | — |

## 6. Interaction States (Hi-Fi)

### Button: [Book a demo]
| State | BG | Text |
|-------|-----|------|
| Default | #6366f1 | #fff |
| Hover | #4f46e5 | #fff |
| Focus | #6366f1 + ring | #fff |

### Service Card (accordion/tab)
| State | Border | Shadow |
|-------|--------|--------|
| Default | 1px #e2e8f0 | none |
| Hover | 1px #e2e8f0 | 0 4px 12px rgba(0,0,0,0.06) |
| Expanded | — | — |

### Testimonial Carousel
| Control | Behavior |
|---------|----------|
| ← → | Prev/next slide |
| Dots | Jump to slide |
| Auto-advance | Optional, ~5s |
