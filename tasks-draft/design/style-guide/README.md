## Design Principles

- **Visual first** — 70% charts/diagrams, 20% captions, 10% text
- **BCG editorial restraint** — light weight 300 headlines, 90% neutral, 10% color
- **One chart per section** — each visual does one job
- **Founder language** — "teams like yours," not "leverage our paradigm"
- **Every stat sourced** — name + year + sample size, no exceptions
